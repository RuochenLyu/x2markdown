# 输出示例

## 普通 post

```md
作者: Example Author (@example)
时间: 2026-03-18 16:30:00
链接: https://x.com/example/status/1234567890123456789

正文:
这是一条示例帖子，包含一个[外部链接](https://example.com)。
第二段正文依然会保留段落换行。

引用内容:
作者: Quoted Author (@quoted)
时间: 2024-01-02 12:55:59
链接: https://x.com/quoted/status/9876543210987654321
正文:
> 这是被引用帖子的正文。
> 
> 这里展示的是复制结果里的引用区块。

图片:
- [图片 1](https://pbs.twimg.com/media/example-1.jpg?format=jpg&name=large)
- [图片 2](https://pbs.twimg.com/media/example-2.jpg?format=jpg&name=large)
```

## X Article

```md
# 用一条长文解释为什么需要一键复制 Markdown

作者: Example Author (@example)
时间: 2026-03-18 17:00:00
链接: https://x.com/example/article/1234567890123456789

正文:
这是文章的第一段正文，用来说明导出的结构会保留标题、作者、时间和链接。

[图片 1](https://pbs.twimg.com/media/example-article-1.jpg?format=jpg&name=large)

这是文章的第二段正文，正文内的普通链接会被转换成[Markdown 链接](https://example.com/article)。
```
